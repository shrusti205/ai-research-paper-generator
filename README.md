# 📚 AI Research Paper Generator

A Flask-based web application that analyzes research papers using Google's Gemini AI.

## ✨ Features

- 📄 Upload PDF research papers
- 🤖 AI-generated summaries
- 🧠 Interactive flashcards
- 🎨 Clean user interface
- ⚡ Powered by Gemini AI

## 🛠 Technologies Used

- Python
- Flask
- HTML
- CSS
- JavaScript
- PyPDF2
- Gemini API

## 🚀 How to Run

Install dependencies:

```bash
pip install -r requirements.txt
```

Run:

```bash
python app.py
```

Open:

```text
http://127.0.0.1:5000
```

## 📸 Project Workflow

1. Upload Research Paper PDF
2. Extract Text
3. Gemini AI Analysis
4. Generate Summary
5. Generate Flashcards
6. Display Results
